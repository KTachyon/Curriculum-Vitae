Curriculum Vitæ
==========
My curriculum vitae typesetted in LaTeX based on the Europass model, based on https://github.com/rfgamaral/Curriculum-Vitae


INSTRUCTIONS
==========
You need XeLaTeX installed on your computer. Command line based compilation is as easy as running:

xelatex cv-europass-en.tex
